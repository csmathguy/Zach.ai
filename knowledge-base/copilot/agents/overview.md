# Agents Overview

- Custom agents are `.agent.md` files with frontmatter: name, description, tools, model, and optional handoffs.
- Use agents to constrain behavior (read-only planning vs. editing) and to preselect tool sets.
- Handoffs coordinate sequential steps (Plan → Implement → Review).

Docs: https://code.visualstudio.com/docs/copilot/customization/custom-agents
