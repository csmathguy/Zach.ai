# Extension Development Overview

- Chat Participant API: create @-mentionable chat participants with commands, follow-ups, and tool calling.
  Docs: https://code.visualstudio.com/api/extension-guides/ai/chat
- Language Model Tool API: contribute tools (function calling) with schemas and confirmations.
  Docs: https://code.visualstudio.com/api/extension-guides/ai/tools
- Consider GitHub App-based Copilot extensions when targeting multiple surfaces.
  Docs: https://docs.github.com/en/copilot/building-copilot-extensions/about-building-copilot-extensions
