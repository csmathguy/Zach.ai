# Tools Overview

- VS Code supports built-in, extension, and MCP tools. Tools are distinct from Agent Skills.
- Enable per-request via tools picker; explicitly invoke in prompts with `#toolName`.
- Group related tools into tool sets for consistency.

- Agent Skills: see [copilot/agent-skills.md](knowledge-base/copilot/agent-skills.md) for portable, task-specific capabilities.
