# Skills (Claude)

Context

- We use “skills” to describe packaged capabilities or repeatable behaviors.
- In Claude’s ecosystem, combine prompts, tools/connectors (MCP partners), and editor integrations to realize skills.

Learn more

- Docs home: https://platform.claude.com/docs
- Claude Code: https://code.claude.com/docs/en/overview
- MCP partners: https://claude.com/partners/mcp

Implementation tips

- Define inputs/outputs clearly; validate parameters before tool calls.
- Keep prompts short and role-specific; prefer composition over monolith prompts.
- Log usage and outcomes to refine effectiveness.
